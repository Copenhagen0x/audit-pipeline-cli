03 — The Product

Four pillars. One adaptive loop.

Four pillars. One adaptive loop. Real-time detection feeds cross-protocol propagation; propagation feeds closed-loop fix delivery; fix delivery feeds the on-chain attestation registry. Every cycle compounds the catalog. Numbers below are live cluster state.

P1
/ Pillar 01 — detection · 27 active

Counterfactual mainnet detection.

For every transaction hitting a covered protocol, a parallel simulation runs against attack-pattern instrumented forks. When counterfactual state diverges from actual, the platform flags the transaction in real time, before the attack chain finalizes. 23-second median from violation event to flagged disclosure. Catches operational and state-mutation classes that point-in-time audits structurally cannot detect.

Stack
RPC subscription LiteSVM forked-state attack-pattern lib divergence detector
P2
/ Pillar 02 — propagation · 41 events

Cross-protocol bug-class propagation.

When a bug is disclosed anywhere — Immunefi, public PR, advisory, or one of our own cycles — the platform auto-extracts the structural attack pattern and searches every indexed protocol for the same class. 6.2-minute median from disclosure to cross-protocol candidate list. F7's pattern alone has been propagated across all 27 targets monthly since the inaugural cycle.

Stack
disclosure feed ingest structural extractor vector embeddings cross-protocol search
P3
/ Pillar 03 — fix delivery · 12.3 PRs/wk

Closed-loop fix bundle.

Confirmed bug → agent generates a candidate fix → Kani harness formally proves the fix preserves every other invariant → full test suite runs → bundle (bug + fix + proof + tests) opens as a single PR. 47-minute median from PoC to PR-ready bundle. Maintainers receive verified fix proposals, not raw findings — the F7 contested-disclosure friction never repeats.

Stack
fix-gen prompt loop Kani synthesizer cargo test runner PR bundler
P4
/ Pillar 04 — attestation · 8,420 on mainnet

On-chain attestation registry.

Every audit cycle publishes a cryptographically-signed Merkle root attesting which invariants were verified at which commit SHA. The registry is on Solana mainnet, queryable, composable. Other protocols — lending, insurance, bridges — can require attestation as a precondition for interaction. ~0.0007 SOL gas per cycle. Apache-2.0.

04 — Methodology

The same loop. Running 24/7.

Four steps. The same pipeline that produced F7 in April 2026 now runs continuously across 27 protocols. Every commit triggers a re-scan. Every disclosure feeds the catalog. The library compounds with every cycle.

/ Step 01

Invariant catalog

3,180 falsifiable claims across 27 protocols. Each entry: target file, line range, severity, conservation property. Library compounds with every confirmed cycle.

/ Step 02

Tool-using verification

Each hypothesis dispatched to a Claude agent with read_file, grep, find_function. Agent walks code paths, renders line-cited verdict. 847 dispatches/day across the cluster.

/ Step 03

Empirical confirmation

Verdicts that need empirical proof escalate. Agent writes a custom cargo test against engine helpers, compiles, runs, classifies PASS/FAIL. 9-minute median.

/ Step 04

Closed-loop fix + attestation

Confirmed finding → Kani-verified fix → bundled PR → on-chain attestation. Every pattern is also auto-propagated across the other 26 protocols. The library compounds.

08 — Where Jelleo fits

STRIDE certifies. Jelleo provides the artifacts.

Solana Foundation · STRIDE · April 6, 2026

STRIDE is the right ecosystem-wide framework.
Jelleo is the artifact-producing layer at the deepest depth available.

STRIDE covers eight pillars — operational security, access controls, multisig, governance, smart-contract integrity, key management, economic design, and threat monitoring. Tier 1 ($10M+ TVL) protocols receive ongoing operations security; Tier 2 ($100M+ TVL) protocols additionally receive formal verification tooling.

Jelleo is built for one specific STRIDE pillar — smart-contract integrity. Where STRIDE evaluates against the framework, Jelleo produces the artifacts: autonomous tool-using agents that read the actual source, render line-cited verdicts, and write Rust PoC tests that compile and run against the engine. The work-product a protocol presents to satisfy STRIDE's smart-contract integrity pillar at the highest available depth.

09 — Inaugural disclosure

F7 — cycle one.

What started here

F7 was cycle one. April 2026. The same tool-using agent pipeline that now runs continuously across 27 protocols was, then, dispatched against a single target: Percolator's mainnet engine. It identified a self-dealing insurance-siphon class — the use_insurance_buffer helper shrinks the insurance counter without debiting the vault, letting the haircut residual grow by exactly the absorbed amount. In a self-trade scenario with bilateral attacker-controlled legs, the K/F-winning side IS the attacker — the grown residual is claimed through the attacker's own winning leg.

What landed

The maintainer closed PR #39 without merging the proposed vault-debit fix. The chosen defense path uses the engine's existing protections — bounded dt, bounded price movement, exact solvency-envelope validation, and the A1 regression suite. On April 28, the disclosure was formally mapped to existing test coverage: tests/test_a1_siphon_regression.rs was relabeled "A1 / PR39-F7 self-dealing insurance-siphon regression tests" and committed to main as a1afd2e. Maintainer verification: 4/4 wrapper paths pass with insurance_drop = 0; Kani harness proof_junior_profit_backing satisfies 2/2 cover properties.

What changed after

F7 was the inaugural cycle. The cluster came online the following quarter. Today the same loop runs against 27 protocols continuously — 184 disclosures cumulative, 8,420 attestations on mainnet. The closed-loop fix bundle pillar that grew out of F7's contested-disclosure friction now ships every PR with a Kani-proven fix that demonstrates it preserves every other invariant before the maintainer ever sees it.

The permanent record

F7 stays on the disclosure ledger as a featured row. Cycle one. Same loop. Different scale.
