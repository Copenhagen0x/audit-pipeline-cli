JELLEO

Colosseum Demo Video Script

3 minutes max · live product walkthrough · no slides · no code · screen + voiceover

Click-by-click, word-by-word recording script. Practice each section out loud twice before the take. Goal isn't to recite — it's to internalize so the language is yours when the recorder is rolling. Six segments, ~30 seconds each, 3:00 total.

Single most important rule: Don't pitch — demonstrate. Every claim is backed by something visible on screen at the moment you say it.

PRE 10 minutes before recording setup

Browser setup:

  • Chrome (clean window, all extensions disabled or hidden, no other tabs)
  • Bookmarks bar hidden (View → Always show bookmarks bar OFF)
  • Browser zoom 100%
  • Window maximized to full screen at 1920×1080 minimum
  • Open jelleo.com in the only tab
  • Hard refresh once (Cmd/Ctrl+Shift+R) so animations start fresh

System setup:

  • External mic if available (USB or XLR — laptop mic OK as fallback)
  • Mac/Windows menu bar hidden if possible (or auto-hide)
  • Phone on Do Not Disturb · Slack / Discord / Mail closed
  • Notifications disabled at OS level
  • Cursor highlight ON (System settings → Accessibility → Cursor size, or use a tool like Mouseposé)

Recording tool:

  • OBS, QuickTime, Loom, or Riverside — any will work
  • Recording resolution: 1920×1080 minimum, 60fps preferred
  • Capture system audio: OFF (avoid YouTube ads / browser sounds bleeding in)
  • Capture mic: ON
Test recording: Do one 30-second test. Play it back. Check audio levels (peak around -6dB), check cursor visible, check no notifications visible. Fix any issue before the real take.

POST Editing notes edits

Cuts to make:

  • Trim leading silence to 0.5 seconds before "This is Jelleo"
  • Trim trailing silence to 1.0 seconds after "occupied"
  • Tighten any pauses longer than 0.7 seconds (except the deliberate ones noted in tactical boxes)
  • Cut any "um", "uh", "you know", repeated words
  • If a section drifts over budget, trim mid-sentence rather than cutting whole lines — keep the structure intact

What NOT to add:

  • No background music (would compete with the technical content)
  • No intro card / title screen (you have 3 minutes — every second on the product, not on a logo bumper)
  • No outro card / "thanks for watching"
  • No subtitles burned in (Colosseum's player will auto-caption)

Final check before upload:

  • Total duration ≤ 3:00
  • Audio peaks at -6dB, no clipping
  • Every URL bar visible matches what you announce
  • No personal info / customer names visible in any frame
  • No browser notification, popup, or extension chrome visible
  • Export at 1080p, MP4, H.264
  • Upload to YouTube as Unlisted · paste link into Colosseum form
